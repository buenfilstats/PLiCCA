Provided is the code to run each of the unsupervised VAE + sparse CCA, deep CCA approaches, PLiCCA-VAE, and PLiCCA-NF approaches.

They can be run on the command line via python run_VAE.py, python run_deep_CCA.py, python run_cond_VAE.py and python run_cond_NF.py respectively, after cding into this "PLiCCA-code" directory. The required packages are numpy, torch, matplotlib, normflows, and Adam_mini.

Before running the models, the synthetic data used in the simulation section of the supplementary materials can be generated by running the cells in the rings_and_discs_data_generation_known_cvs.ipynb notebook.

Note that, to use PLiCCA-NF (via "python run_cond_NF.py"), first you must run the unsupervised VAE (using "python run_VAE.py"), and then run all the cells in the "rings_and_discs_post_run_VAE_analysis" notebook (the last cell saves the dimension-reduced unsupervised VAE latent space to be used as the input data to PLiCCA-NF).

After running the models, you can see the training results in the corresponding notebooks. For ease of use, we recommend not changing the directory structure.
